# TimeWeaver - 바쁜 일상, 스마트한 관리

### 2024-1 모바일프로그래밍 기말 프로젝트

## 🖥️ 프로젝트 소개
`TimeWeaver`는 바쁜 현대인들에게 맞춤형 스케줄표를 제공해 제한 시간에 할 일을 모두 마칠 수 있도록 해주는 서비스입니다. <br><br>
